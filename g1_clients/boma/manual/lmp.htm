Load Management

 

Load Management consists of 2 Sections :

Selection | Profile Report

Section 1: LMP and Tenant Selection

In this section users select the type of profile they wish to view and the tenant they wish to view a particular profile for.

The elements and actions invoked are as follows:

SELECTIONS

    1. Select Tenant - Drop this selection box to select a tenant
    2. Meter - once a tenant is selected, this list will be populated with the meters found for that tenant. Drop this selection box to select a specific meter for the selected tenant
BUTTONS
    1. Building LMP- this button will bring up the buildings LMP showing the last available day that data was collected for. For gEnergyOne enabled buildings, this would show todays load profile on an hourly scale.
    2. Meter LMP - after a tenant is selected and the meter list is populated, this button will show the available load profile data for the meter shown.For gEnergyOne enabled meters, this would show todays load profile for the meter on an hourly scale. note: this action is automatically invoked when a new meter is selected off the list.
    3. Aggregate LMP - after a tenant is selected, this button will show the available aggregated data for all the tenants meters. note: because the data shown may be an aggregation of both gEnergyOne enabled meters and traditional meters, the last date of available data is based on the last date of data collected on the traditional meters. A greater level of data may exist on the gEnerrgyOne enabled meters

Section 2: Profile Report

The Profile Report provides extensive support for how you may wish to view your report and navigation

The Profile Report breaks down to 5 sections:

Date Scrolling | Graph | Zoom Panel | Peaks | History

Date Scrolling

As seen in the image below, left and right navigation buttons are located directly above the graph. These buttons allow the user to move through available data for dates available.

Graph

Your LMP graph will show you KW demand in hourly format. For gEnergyOne enabled LMP's it's also possible to view 30 & 15 minute interval data in realtime.

If viewing aggregated LMP's, it's possible to click on legend items to view more detailed reports.

 

Zoom Panel

The Zoom Panel allows users to specify date, time range, interval, and graph format to get a better perspective on usage data.

Peaks / Consumptions Table

The peaks / consumption table shows users the peak demand and current consumption for the period the graph is reporting on.

Peaks / Consumptions History Table

The peaks / consumption history table shows users the peak demand and consumption history by period. Hyperlinks provide easy access to peak demand days.
