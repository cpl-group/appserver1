Portfolio Cost Analysis

 

Portfolio Cost Analysis consists of 2 sections :

Selection | Report Window

Section 1 : Building / Year / Report Selector

In this section users select the building, years the users wishes to compare, and the report they wish to view.

The elements and actions invoked are as follows:

SELECTIONS

    1. Select Buildings- drop this selection box to select a building
    2. Year 1 / Year 2 - once a building is selected, these 2 list boxes will be populated with the years available for reporting. Select 2 years for comparison by selecting a year in each drop box.
    3. Report - drop this selection box to select the report you wish to view.

BUTTONS

    1. View - click this button to load the desired options into the Report Window.

Section 2: Report Window

The report window displays the report as detailed in the selection area.
