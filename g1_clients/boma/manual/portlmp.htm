Portfolio Load Management

 

Portfolio Load Management consists of 2 Sections :

Selection | Profile Report

Section 1: LMP and Building Selection

In this section users view the Portfolio LMP or select a building and view it's LMP report.

The elements and actions invoked are as follows:

SELECTIONS
      1. Select Building - Drop this selection box to select a building.Once the building is selected, the LMP report will be loaded.
BUTTONS
      1. Portfolio LMP- this button will show the available aggregated data for all the buildings in the Portfolio. note: because the data shown may be an aggregation of both gEnergyOne enabled buildings and traditional buildings, the last date of available data is based on the last date of data collected on the traditional building. A greater level of data may exist on the gEnerrgyOne enabled buildings

Section 2: Profile Report

The Portfolio Profile Report follows the same functionality as the building level LMP's
