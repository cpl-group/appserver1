Meter Services

 

Meter services consists of 2 sections :

Selection | Invoice Window

Section 1 : Tenant / Period Selection

In this section users select the type of profile they wish to view and the tenant they wish to view a particular profile for.

The elements and actions invoked are as follows:

SELECTIONS

    1. Select Tenant - Drop this selection box to select a tenant
    2. Meter - once a tenant is selected, this list will be populated with the periods available to bill for the tenant selected. Drop this selection box to select a specific period. After selecting the period, the invoicee will be loaded in the Invoice Window.

BUTTONS

    1. Print Invoice - this button will print the invoice displayed in the Invoice Window.

 

Section 2: Invoice Window

The invoice window displays the invoice for the selected tenant and period
